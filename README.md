# The Laurie Fix

A browser-based helper that makes covered AO3 EPUBs behave properly on Kobo Libra Colour.

## Use

1. Add a cover to the AO3 EPUB using Alex Chan's cover tool.
2. Open The Laurie Fix website.
3. Choose one or more covered EPUB files.
4. Download the files ending in `Kobo Ready.epub`.
5. Put those files in the Rakuten Kobo folder in Google Drive.

The book conversion happens locally in the browser. Books are not uploaded to a server.

Created because Laurie refused to accept ugly book covers.

Organized. Pretty. Fixed.
